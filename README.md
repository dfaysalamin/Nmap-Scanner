
# Nmap Automation Scanner

Interactive bash script for automated Nmap scanning.

## Features
- Multiple scan types (TCP, SYN, UDP, Xmas, Null)
- Port range selection
- OS detection
- Multiple output formats
- Automatic timestamp

## Usage
chmod +x nmap_scanner.sh
sudo ./nmap_scanner.sh
